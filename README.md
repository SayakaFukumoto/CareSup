# CareSup
### 概要💡
CareSupは介護職等の記録業務の手伝いをします。
運営・看護職員・介護職員の記録をまとめ、一つの画面で表示することができます。
また、ご入居者様の情報を抽出し、一人ひとりの詳細情報も表示できます。

今まで紙に手書きで記入していた施設は、このシステムの導入により
数々の業務記録の記録・確認をスムーズに行うことができます。 
  
 
### 使い方 🐚

--composerをインストール。

--```composer install```を実行すると、venderフォルダ等必要なものがインストールされます。
 
--プロジェクト直下の.env.sampleファイルをご自身の環境に合わせて編集し、ファイル名を.envに変更。<br>
　 config/database.phpのDB情報もご自身の環境のものに更新。

--dompdfの環境設定。vendor\dompdf\dompdf\src\FrameReflower\Text.phpで$_wordbreak_pattern= '//u'に変更。

--コンソールにて```php artisan migrate```,```php artisan db:seed ```,```php artisan storage:link```を実行していただきますと、データベースの作成・テストデータの挿入、画像入力が可能になるかと思います。(使用DB名:care_sup)


#### テストアカウント
【ログインID】

test1 or test2  …　ホーム長/運営(同権限)

test3 or test4  …　看護職員/介護職員（ほぼ同権限）

【初期パスワード】
test123


#### 各権限でできること
介護職員     　…　全記録(お知らせ・共有事項・看護記録・介護記録)の閲覧。入居者情報の閲覧・一部編集。共有事項・業務記録の入力。自身が作成した記録の削除。

看護職員　　　 …　上記に加え看護記録の入力。　


ホーム長/運営　…　上記に加え新規ユーザー登録と入居者登録。お知らせの入力・編集。全記録の削除。入居者情報編集・削除。
